- `feed_id >= 0` (regular feed): count unread in ttrss_user_entries WHERE feed_id=X
- `feed_id == -1` (starred): count WHERE unread=True AND marked=True

- Negative feed_id triggers `ccache_update_all` for label recalculation (ccache.php:110-113)

## Batch 8: `ttrss/tasks/feed_tasks.py` (article persistence — close TODO)

Source: `ttrss/include/rssfuncs.php:545-1117`

Full scope (R16):

### 1. GUID Construction (rssfuncs.php:550-560)
- Fallback chain: `item.get_id()` → `item.get_link()` → `make_guid_from_title(item.get_title())`
- Owner-scoped: `f"{owner_uid},{entry_guid}"`
- Hashed: `"SHA1:" + hashlib.sha1(entry_guid.encode()).hexdigest()`
- Truncated to 245 chars (rssfuncs.php:621)

### 2. Content Hash (rssfuncs.php:707)
- `"SHA1:" + hashlib.sha1(entry_content.encode()).hexdigest()`

### 3. Entry INSERT/UPDATE (rssfuncs.php:720-750, 956-962)
- INSERT into ttrss_entries if new GUID
- UPDATE date_updated if existing

### 4. User Entry INSERT (rssfuncs.php:887-894)
- INSERT into ttrss_user_entries with filter-determined unread, marked, published, score

### 5. Filter Actions (rssfuncs.php:812-828, 845-863)
- All 8 types via `from ttrss.articles.filters import load_filters, get_article_filters, find_article_filter, calculate_article_score, assign_article_to_label_filters`

### 6. Tag Extraction + tag_cache (rssfuncs.php:634-648, 1062-1097)
- Parse `<category>` from feed items
- Add manual tags from filter actions
- Filter blacklisted tags
- Save to ttrss_tags
- Write tag_cache column

### 7. Label Auto-Assignment (rssfuncs.php:1102-1113)
- If AUTO_ASSIGN_LABELS pref, match label captions against tags+content+title

### 8. Enclosures (rssfuncs.php:982-1020)
- INSERT into ttrss_enclosures (deduplicated by content_url + post_id)

### 9. N-gram Dedup (rssfuncs.php:867-882)
- PostgreSQL `similarity()` function for _NGRAM_TITLE_DUPLICATE_THRESHOLD
- Mark as read if similar article found

### 10. Content Update Detection (rssfuncs.php:923-970)
- Compare content_hash, title, num_comments, plugin_data
- If changed: UPDATE ttrss_entries, optionally mark as unread
